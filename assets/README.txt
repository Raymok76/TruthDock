Assets Folder - Required Image Files
=====================================

Please add the following image files to this folder:

1. trump_icon.jpg
   - Donald Trump profile picture/icon
   - Recommended size: 200x200 pixels
   - Format: JPG or PNG
   - Used as: Poster avatar in the post cards

2. truth_logo.png
   - Truth Social logo (the blue "TRUTH." branding)
   - Recommended size: 300x100 pixels (wide format)
   - Format: PNG with transparent background preferred
   - Used as: Main branding logo in header section

If these files are not provided, the website will still work but will show broken image icons.

Where to get these images:
- Use official Truth Social branding materials
- Ensure you have proper usage rights
- Optimize images for web (compress to reduce file size)
